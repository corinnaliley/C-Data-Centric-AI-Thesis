TASK:
Schreiben Sie eine Funktion 

```
char* string_filter(const char* text)
```  

die einen Zeiger auf einen String entgegennimmt, entsprechend der Anforderungen filtert und das Ergebnis zurückgibt.

*Anforderungen*

- Der Rückgabestring soll nur noch Kleinbuchstaben und Großbuchstaben enthalten.

### Beispielaufruf
```
string_filter("...Hallo Welt!!");
```

### Beispielergebnis
```
HalloWelt
```


SOLUTION:
#include <string.h>
#include <stdlib.h>
char* string_filter(const char* text){
  size_t textlen = strlen(text);
  char* return_text = malloc((textlen+1)*sizeof(char));
  int pos = 0;
  for(int i = 0; i < textlen; i++){
    char c = text[i];
    if((c >= 'A' && c <= 'Z')|| (c >= 'a' && c <= 'z')){
      return_text[pos] = text[i];
      pos++;
    }
  }
  return_text[pos] = '\0';
  return return_text;
}
