TASK:
Programmieren Sie eine Funktion mit folgendem Prototyp:

```
int frequency(const char string[], char ch);
```
 
*Anforderungen*
 
- Die Funktion bekommt im Parameter `ch` ein Zeichen und im Parameter `string` einen Zeiger auf ein Feld, in dem ein String gespeichert ist, übergeben.
- Die Funktion bestimmt wie häufig das übergebene Zeichen im String enthalten ist und liefert diese Anzahl zurück.
- Bei Buchstaben wird nicht unterschieden, ob es sich um Klein- oder Großbuchstaben handelt. <u>Beispiel</u>. Im String `Test!` ist sowohl das Zeichen `T` als auch das Zeichen `t` zweimal enthalten. 
- Benutzen Sie zur Erkennung und Umwandlung der Zeichen Funktionen aus der Standardbibliothek.
- Gehen Sie davon aus, dass `string` entweder der `NULL`-Zeiger ist oder auf den Anfang eines gültigen Strings zeigt.


SOLUTION:
#include <ctype.h>
#include <string.h>
 
int frequency(const char string[], char ch){
  if(string == NULL){
    return 0;
  }
  int count = 0;
  for(int i = 0; i< strlen(string); i++){
    if(tolower(ch) == tolower(string[i])){
      count++;
    }
  }
  return count;
}
