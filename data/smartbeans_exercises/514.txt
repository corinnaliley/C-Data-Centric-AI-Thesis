TASK:
Programmieren Sie eine Funktion mit folgendem Prototyp

```
int *bucket_sort( int values[], int length, int max );
```

die das Feld der Länge `length`, auf dessen Anfang `values` zeigt, sortiert. Dafür darf der Wert von
`max` nicht negativ sein und das Feld darf nur Werte zwischen `0` und `max` enthalten.

Sortieren Sie wie folgt:

1. Speicher für ein `int`-Feld der Länge `max+1` auf dem Heap anfordern (hier kann ein Fehler auftreten) und initialisieren jeder Feldkomponente mit `0`.
2. Durchlaufen von `values`.
    - Wird der Wert `x` gelesen, dann wird der Wert an der Position `x` im `int`-Feld auf dem Heap um `1` erhöht.
    *Motivation*: In dem `int`-Feld auf dem Heap wird für jeden Wert von `0` bis `max` abgespeichert, wie oft dieser Wert in `values` vorkommt.
3. Durchlaufen des `int`-Feldes auf dem Heap, dabei wird `values` neu geschrieben, beginnend bei Position `0`.
    - Wird an der Position `x` im `int`-Feld auf dem Heap der `n` Wert gelesen, muss in `values` `n`-mal hintereinander `x` geschrieben werden.
    *Hinweis*: Die Einfügeposition in das Feld `values` muss verwaltet werden.

*Anforderungen*

- Die Funktion liefert bei einem Fehler den Nullzeiger, sonst einen Zeiger auf das sortierte Feld zurück.
- Sie brauchen nicht zu prüfen, ob `values` auf den Anfang eines `int`-Feld zeigt und dieses Feld die Länge `length` hat.
- Prüfen Sie, ob `max` und das Feld, auf dessen Anfang `values` zeigt, zulässige Werte enthält.
- Vor Beendigung der Funktion wird angeforderter und nicht mehr benötigter Speicher auf dem Heap wieder freigeben.

### Beispielaufruf
```
int a[] = {5,3,6,1,5,4,8,9,6,6,6};
int *result = bucket_sort(a, 11, 9);
```
### Beispielergebnis
```
result = {1,3,4,5,5,6,6,6,6,8,9}
```

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int *bucket_sort(int values[], int length, int max);

int *bucket_sort(int values[], int length, int max) {
  if (max < 0) {
    return NULL;
  }
  for (int i = 0; i < length; i++) {
    if (values[i] < 0 || values[i] > max) {
      return NULL;
    }
  }
  int *bucket_array;
  bucket_array = (int *)calloc(max + 1, sizeof(int));
  if (bucket_array == NULL) {
    return NULL;
  }
  for (int i = 0; i < length; i++) {
    bucket_array[values[i]] += 1;
  }
  int *start = values;
  for (int i = 0; i < max + 1; i++) {
    for (int j = 0; j < bucket_array[i]; j++) {
      *start = i;
      start++;
    }
  }
  free(bucket_array);
  return values;
}
