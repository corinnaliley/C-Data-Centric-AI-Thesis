TASK:
Das Busunternehmen "Neumann" möchte eine Statistik über das Fahrgastverhalten erheben. Es ist daran interessiert, wie viele Fahrgäste sich an jeder Haltestelle im Bus befinden.
Schreiben Sie eine Funktion 
```
int* fahrgaeste(int einaus[][2], int n)
```

Diese nimmt ein zweidimensionales Feld `einaus[n][2]` sowie die Anzahl der Haltestellen entgegen. 
Jede Zeile dieses Arrays steht für eine der `n` Bushaltestellen. In der Spalte 0 wird gespeichert, wie viele Personen an dieser Station einsteigen. In Spalte 1 steht, wie viele Menschen aussteigen.
 
Die Funktion soll ein eindimensionales Feld der Länge `n` zurückgeben, in dem für jede Bushaltestelle festgehalten wird, wie viele Menschen sich im Bus befanden, nachdem alle ein- und ausgestiegen sind. Zu Beginn einer jeden Fahrt ist der Bus leer.
 
*Hinweise*

- Verwenden Sie für die Speicherallokation in dieser Aufgabe die Funktion `malloc()`.

### Beispielaufruf
```
int einaus[5][2] = {{5, 0},{4, 2},{5, 1},{5, 2},{0, 1}};
int* gaeste = fahrgaeste(einaus, 5);
```
 
### Beispielergebnis
```
{5, 7, 11, 14, 13}
```


SOLUTION:
#include <stdlib.h>
 
int* fahrgaeste(int einaus[][2] , int n){
 
  int* a = malloc(n*sizeof(int));
  int guests = 0;
  for(int i=0; i<n; i++){
      guests += einaus[i][0];
      guests -= einaus[i][1];
      a[i] = guests;
  }
  return a;
 
}
