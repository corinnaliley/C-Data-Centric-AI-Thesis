#include <stdio.h>

int main() {
  int n;
  scanf("%d", &n);

  int sum = 0;
  int sign = 1;
  while (n > 0) {
    sum += sign * n % 10;
    n /= 10;
    sign *= -1;
  }

  printf("%d\n", sum);

  return 0;
}

Die Quersumme einer Zahl ist die Summe der einzelnen Ziffern.
Die alternierende Quersumme einer Zahl wird gebildet indem man die einzelnen Ziffern der Zahl abwechselnd subtrahiert und addiert. Dabei beginnt man von rechts mit der Einerstelle, also der letzte Stelle/Ziffer. 

*Beispiel*.  aQS(35916) = 6-1+9-5+3 = 12

Schreiben Sie ein Programm, dass eine Zahl > 0 einliest und deren alternierende Quersumme ausgibt.

### Beispieleingabe
```
18
214493
```
### Beispielausgabe
```
7
-7
```
