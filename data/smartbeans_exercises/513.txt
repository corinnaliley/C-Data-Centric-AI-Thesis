TASK:
Schreiben Sie eine Funktion mit folgendem Prototyp 

```
int lex_sorted( const char* array[] );
```

Im Feld `array` werden Strings (genauer: Zeiger auf die erste Speicherstelle von `char`-Feldern) übergeben, 
der Wert der letzten Feldkomponente von `array` ist der Null-Zeiger.
Die Funktion prüft, ob die Strings im Feld `array` lexikographisch aufsteigend sortiert
sind. Der Funktionswert ist `1` wenn dies der Fall ist, sonst `0`.

Verwenden Sie für die lexikographischen Vergleiche der Strings die Funktion `strcmp`, die
mit folgendem Prototyp in der Standardheaderdatei `string.h` deklariert ist.

```
int strcmp( const char *s1, const char *s2 );
```

Die Funktion liefert einen `int`-Wert kleiner, gleich oder größer Null, je nachdem, ob der
String, auf dessen Anfang `s1` zeigt, lexikographisch kleiner, gleich oder größer ist als der
String, auf dessen Anfang `s2` zeigt.

*Hinweise*

- Sie brauchen nicht zu prüfen, ob `array` verschieden von `NULL` ist, dass ist beim Aufruf der Funktion bereits geklärt.
- Achten Sie auf die Grenzfälle, z.B. leeres Feld oder Feld enthält nur ein Element.
- Die Kenntnis der von `strcmp` verwendeten lexikographischen Ordnung ist für das Lösen der Aufgabe nicht erforderlich.

### Beispielaufruf
```
const char* s[5];
s[0] = "1";
s[1] = "4";
s[2] = "test";
s[3] = "2";
s[4] = NULL;

lex_sorted(s)
```
### Beispielergebnis
```
0 
```

#include <stdio.h>
#include <string.h>

int lex_sorted(const char * array [] ) {
  int i ;

  /* Grenzfaelle */
  if ( array [0] == NULL || array [1] == NULL )
    return 1;
  
  /* Ueberpruefung */
  for ( i = 1; array [ i ] != NULL ; i ++ ) {
    if ( strcmp ( array [i -1] , array [ i ]) > 0 )
    return 0;
  }
  return 1;
}
