TASK:
Die Mittquadratmethode ist eine einfache Methode zur Erzeugung von Pseudozufallszahlen. Ausgehend von einer Ausgangszahl wird wiederholt das Quadrat der inneren Ziffernkette gebildet.

Entwickeln Sie eine Funktion
```
int *middle_square(int seed)
```
, welche die die Mittquadratmethode implementiert.

Anforderungen:
- Die Funktion nimmt eine zweistellige Zahl als seed (Startwert) entgegen und gibt ein Feld der Länge 100 mit den Werten der Folge zurück
- Der Seed ist der erste Wert in der Liste, alle weiteren werden entsprechend der Bildungsvorschrift bestimmt
- Für Startwerte welche mehr oder weniger als 2 Stellen haben gibt die Funktion den Nullzeiger zurück


SOLUTION:
#include <stdio.h>
#include <stdlib.h>
#include <math.h>

int *middle_square(int seed);
int *middle_square(int seed) {
  int squared;
  int current = seed;
  int *liste = calloc(sizeof(int), 100);
  for(int i = 0; i < 100; i++) {
    // Calculate squared value of current seed
    squared = current * current;
    liste[i] = current;

    if(squared < 1000) {
      current = squared / 10;
    } else {
      current = (squared / 10) % 100;
    }
  }
  return liste;
}
