#include <stdio.h>

int main() {
  int n;
  scanf ("%d", &n);

  int sum = n;
  while (sum > 9) {
    n = sum;
    sum = 0;
    while (n > 0) {
      sum += n % 10;
      n /= 10;
    }
  }
 
  printf("%d\n", sum);

  return 0;
}

TASK:
Die Quersumme einer Zahl ist die Summe der einzelnen Ziffern.
Bei der iterierten Quersumme einer Zahl wird so lange immer wieder die Quersumme berechnet bis diese einstellig ist.

*Beispiel*.  iQS(35916) = iQS(3+5+9+1+6) = iQS(24) = iQS(2+4) = 6

Schreiben Sie ein Programm, dass eine Zahl > 0 einliest und deren iterierte Quersumme ausgibt.

### Beispieleingabe
```
18
214493
```
### Beispielausgabe
```
9
5
```
