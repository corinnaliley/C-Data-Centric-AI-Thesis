TASK:
Eine Ampel kann als fortlaufende Zahl an Zeiteinheiten dargestellt werden. Die Ampel beginnt in der `ROT` Phase bei der Zeiteinheit 0. Jedesmal wenn die Zeiteinheit um eins vergrößert wird schaltet die Ampel eine Phase weiter. Die Phasen werden in folgender Reihenfolge durchgeschalten: `ROT`, `ROT/GELB`, `GRÜN`, `GELB`.

Schreiben Sie ein Programm, das eine Zahl >= 0 einliest und ausgibt in welcher Phase sich die Ampel zu diesem Zeitpunkt befindet.

### Beispieleingabe
```
4
18
21
3
```
### Beispielausgabe
```
ROT
GRÜN
ROT/GELB
GELB
```
