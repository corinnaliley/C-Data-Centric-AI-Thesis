TASK:
Der Datentyp `interval_t` zur Beschreibung von abgeschlossenen Intervallen ganzer Zahlen durch die untere 
und obere Intervallgrenze, ist wie folgt gegeben.

```
typedef struct {
  long low;   // lower bound
  long up;    // upper bound
} interval_t;
```

Ein Intervall ist genau dann leer, wenn seine untere Intervallgrenze größer ist als seine obere. 
Der Durchschnitt zweier Intervalle ist wieder ein (möglicherweise leeres) Intervall.

Programmieren Sie eine Funktion, die bestimmt ob der Durchschnitt der übergebenen
Intervalle leer ist, mit folgendem Prototyp.

```
int is_empty_intersection(const interval_t *a, const interval_t *b);
```

*Anforderungen:*

- Das Durchschnittsintervall der Intervalle, auf die a und b verweisen, wird bestimmt.
- Ein übergebener Null-Zeiger wird wie ein leeres Intervall behandelt.
- Die untere Intervallgrenze des Durchschnittsintervalls ist gegeben durch das Maximum der beiden unteren Intervallgrenzen, die obere Intervallgrenze durch das Minimum der beiden oberen Intervallgrenzen.
- Die Funktion liefert 0 zurück wenn das Durchschnittsintervall nicht leer ist, sonst wird ein positiver Wert zurückgeliefert.
- Definieren Sie die Struktur `interval_t` in der Abgabedatei.


SOLUTION:
#include <stdlib.h>

typedef struct {
  long low;   // lower bound
  long up;    // upper bound
} interval_t;

int is_empty_intersection(const interval_t *a, const interval_t *b) {
  if (a == NULL || a->low > a->up || b == NULL || b->low > b->up)
    return 1;

  interval_t inter;

  /* max lower bound */
  if (a->low > b->low)
    inter.low = a->low;
  else
    inter.low = b->low;

  /* min upper bound */
  if (a->up < b->up)
    inter.up = a->up;
  else

    inter.up = b->up;

  /* ( upper < lower ) <= > empty */

  return (inter.up < inter.low);
}
