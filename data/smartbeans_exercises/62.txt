TASK:
Rufen Sie in der Konsole den Befehl `factor 144` auf. Die Ausgabe 
```
144: 2 2 2 2 3 3
```
gibt die Primzahlzerlegung von 144 an. Programmieren Sie dies in C nach, um beliebige ganze Zahlen bis zur Größe `UINT MAX` verarbeiten zu können. Die zu verarbeitende Zahl soll mit scanf eingelesen werden und die Primfaktoren sollen in aufsteigender Reihenfolge ausgegeben werden.

### Beispieleingabe
```
144
```
### Beispielausgabe
```
144: 2 2 2 2 3 3
```


SOLUTION:
#include <stdio.h>

int main(void) {
  unsigned int zahl;
  scanf("%i", &zahl);
  printf("%i: ", zahl);

  int i = 2;
  while (i < zahl) {
    while (zahl % i != 0) {
      i++;
    }
    while (zahl % i == 0) {
      zahl /= i;
      printf("%i ", i);
    }
  }
  return 0;
}
