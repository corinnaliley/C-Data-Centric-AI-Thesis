TASK:
Diese Aufgabe baut auf der Aufgabe *String Basics I* auf. Sie sollen üben, wie man mit den Funktionen `scanf()`und `getchar()` Strings einlesen kann.
Schreiben Sie dafür ein Programm mit folgender Funktion:
 
- Der Nutzer kann als erstes den Namen eines Empfängers eingeben. Dieser besteht aus einem zusammenhängenden Wort und ist maximal 30 Zeichen lang. Nutzen Sie zum Einlesen `scanf()`.
- Anschließend kann der Nutzer eine Botschaft eingeben. Diese kann mehrere Zeilen umfassen, ist aber maximal 500 Zeichn lang. Nutzen Sie zum Einlesen `getchar()`.
- Zuletzt gibt das Programm Empfänger und Nachricht so aus, wie es in der Beispielausgabe zu sehen ist.
 
### Beispieleingabe
```
Bohnifatius
 
Lieber Bohnifatius,
ich hoffe es geht dir gut!
Grüße,
Bohnita
```
 
### Beispielausgabe
```
Empfänger: Bohnifatius
Nachricht:
Lieber Bohnifatius,
ich hoffe es geht dir gut!
Grüße,
Bohnita
```
 
*Hinweise.*
 
- Wenn Sie diese Aufgabe gelöst haben, können Sie als nächstes die Aufgabe *String Basics III* lösen.


SOLUTION:
#include <stdio.h>
#include <stdlib.h>
 
int main(){
  char empf[30];
  char msg[500];
  int i = 0;
  scanf("%s", empf);
  while(i<500 && (msg[i] = getchar()) != EOF){
    i++;
  }
  msg[i] = '\0';
  printf("Empfänger: %s\nNachricht:%s", empf, msg);
}
