TASK:
Diese Aufgabe baut auf der Aufgabe *String Basics II* auf. 
Sie sollen üben, wie man Strings an eine Methode übergeben und zurückgeben kann.
 
Schreiben Sie hierfür zwei Methoden:
 
```
char* static_str(char* string)
```   
nimmt einen Zeiger auf ein Char-Feld `string` entgegen.
Legen Sie ein statisches Array der Länge 150 an und kopieren Sie `string` mit Hilfe der Funktion `strcpy()` dort hinein.
Geben Sie anschließend einen Zeiger auf die Anfangsadresse des Arrays zurück.
 
Zu Lernzwecken können Sie auch ausprobieren was passiert, wenn das Array innerhalb der Funktion nicht statisch ist.
 
---

```
char* malloc_str(char string[], int len)
```
nimmt ein char-array `string` entgegen, sowie dessen Länge `len`.
Allokieren Sie mit `malloc()` ausreichend Speicherplatz und kopieren Sie den Inhalt von `string` dorthin.
Geben Sie dann einen Zeiger auf den neu erstellten String zurück.
 
### Beispielaufruf
 
```
char* str1 = static_str("Möglichkeit 1");
 
char str2[] = "Möglichkeit 2 (schöner)";
char* str2ptr = malloc_str(str2, 24);
```
 
### Beispielergebnis
 
```
printf("%s", str1);
>>> Möglichkeit 1
 
printf("%s", str2ptr);
>>> Möglichkeit 2 (schöner)
```


SOLUTION:
#include <string.h>
 
char* static_str(char* string){
  static char str_return[150];
  strcpy(str_return, string);
  return str_return;
}
 
char* malloc_str(char string[], int len){
  char* ptr = malloc(len*sizeof(char));
  strcpy(ptr, string);
  return ptr;
}
