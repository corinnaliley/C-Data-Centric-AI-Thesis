TASK:
Programmieren Sie eine Funktion 
```
int suche(int *array, int len, int value);
```
die ein absteigend sortiertes Array `array`, die Länge des Arrays `len` und einen gesuchten Wert `value` übergeben bekommt. 

*Anforderungen*
- Die Funktion gibt `1` zurück, wenn `value` in `array` vorhanden ist und `0`, wenn `value` nicht in `array` vorhanden ist. 
- Die Suche erfolgt **iterativ** und nach dem Prinzip *devide and conquer*. 
- Dafür wird zuerst das mittlere Element von `array` betrachtet. 
- Ist der Wert gleich dem Wert von `value`, so ist die Suche beendet. 
- Ansonsten wird das Array in zwei Hälften aufgeilt: 
- War `value` kleiner als der Mittelwert? 
- Dann suche im rechten Teil der Liste weiter, ansonsten führe die Suche im linken Teil fort.

### Beispielaufruf
```
int a[] = {100,80,60,50,44,20,10};
suche(a, 7, 20);
```

### Beispielergebnis
```
1
```


SOLUTION:
#include <stdio.h>

int suche(int *array, int len, int value) {
  int low = 0, high = len-1;
  while (low <= high) {
    int mitte =  low + ((high - low) / 2);
    if (array[mitte] == value) {
      return 1;
    }
    if (value > array[mitte]) {
      high = mitte - 1;
    } else {
      low = mitte + 1;
    }
  }
  return 0;
}
