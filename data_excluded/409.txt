TASK:
Schreiben Sie eine Funktion 

```
char* simpleTranslate(char* text, translation_t* dictionary, size_t dict_size)
```

welche die Funktion eines einfachen Übersetzers implementiert. Die Funktion bekommt die folgenden Argumente übergeben:
 
- `char* text` Ein Zeiger auf den Text, der übersetzt werden soll.
- `translation_t* dictionary` Ein Zeiger auf ein Feld aus `translation_t`-Objekten, die ein Wörterbuch repräsentieren.
- `size_t dict_size` Die Anzahl der `translation_t`-Elemente im Dictionary.
 
Ein `translation_t`-Element hat folgende Struktur:
 
```
typedef struct {
  char* src; // Ausgangswort
  char* dest; // Zielwort (Übersetzung)
} translation_t;
```
 
*Anforderungen*
 
- Die Funktion gibt einen Zeiger auf einen String zurück, in dem jedes Wort entsprechend des `dictionary` übersetzt wurde.
- Sie können davon ausgehen, dass der Ausgangstext nur `char`-Zeichen und Leerzeichen enthält.
- Sie können davon ausgehen, dass der übersetzte Text nicht länger als 500 Zeichen ist.
- Der Rückgabestring muss auf dem Heap gespeichert sein. Allokieren Sie also Speicher mit `malloc()`.
- Definieren Sie die Struktur `translation_t` in der Abgabedatei.   
 
*Hinweise*
  
- Diese Aufgabe ist ziemlich kniffelig! Hilfreich sind die Funktionen <a target="_blank" rel="noopener noreferrer" href="https://www.tutorialspoint.com/c_standard_library/c_function_strcat.htm">**strcat**</a>, <a target="_blank" rel="noopener noreferrer" href="https://www.tutorialspoint.com/c_standard_library/c_function_strcpy.htm">**strcpy**</a> und <a target="_blank" rel="noopener noreferrer" href="https://www.tutorialspoint.com/c_standard_library/c_function_strtok.htm">**strtok**</a>. 
 
### Beispielaufruf
```
translation_t dictionary[2] = {{"Hallo", "Hello"}, {"Welt", "World"}};
char* text_trans = simpleTranslate("Hallo Welt", dictionary, 2);
```
 
### Beispielergebnis
```
text_trans = "Hello World"
```

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct {
  char *src;
  char *dest;
} translation_t;

const char *translateWord(char *word, translation_t *dictionary,
                          size_t dict_size) {
  char *translation;
  for (int i = 0; i < dict_size; i++) {
    if (strcmp(dictionary[i].src, word) == 0) {
      return dictionary[i].dest;
    }
  }
  return NULL;
}

char *simpleTranslate(char *text, translation_t *dictionary, size_t dict_size) {
  char *trans_text = malloc(500 * sizeof(char));
  if (trans_text == NULL) {
    printf("Keinen Speicher bekommen");
    return NULL;
  }
  strcpy(trans_text, "\0");
  char *token = strtok(text, " ");
  int i = 0;
  while (token != NULL) {
    const char *trans_token = translateWord(token, dictionary, dict_size);
    if (trans_token == NULL) {
      return NULL;
    }
    if (i != 0) {
      strcat(trans_text, " ");
    }
    strcat(trans_text, trans_token);
    token = strtok(NULL, " ");
    i++;
  }
  return trans_text;
}
