TASK:
Malu und Luca schreiben SMS im Schulunterricht. Sie haben sich beide eine eigene Codesprache überlegt, die sie strikt befolgen.
Malu versteht nur Nachrichten, in denen jeder zweite Buchstabe groß geschrieben ist, beginnend mit einem Kleinbuchstaben, zum Beispiel:

- `hAlLo MaLu WiE gEhT eS dIr`
- `hDgDl`

Luca versteht nur Nachrichten, in denen jeder zweite Buchstabe klein geschrieben ist, beginnend mit einem Großbuchstaben, zum Beispiel:

- `HaLlO lUcA wIe GeHt Es DiR`
- `HdGdL`

Schreiben Sie eine Funktion 

```
char* convert_message(const char *message, char receiver)
```

Diese nimmt eine Nachricht `message` und einen Empfänger `receiver` (`'M'` oder `'L'`) entgegen, und gibt die Nachricht so codiert zurück, dass der Empfänger sie lesen kann. 

*Anforderungen*

- Die Nachrichten enthalten nur Buchstaben und Leerzeichen.
- Die Leerzeichen gelten nicht als Buchstaben und werden für die Zählung übersprungen.
- Wenn der Empfänger nicht `'M'` oder `'L'` ist, soll die Funktion `Mit dir schreib ich nicht!` zurückgeben.


SOLUTION:
#include <string.h>
#include <stdlib.h>
#include <ctype.h>

char* convert_message(const char *message, char receiver){
  size_t len = strlen(message)+1;
  char *new_message = malloc(len*sizeof(char));
  int upper;
  if(receiver == 'M') {
    // 0 = lowercase 1 = uppercase
    upper = 0;
  }
  else if (receiver == 'L'){
    upper = 1;
  }
  else{
    return("Mit dir schreib ich nicht!");
  }
  for(int i = 0; i < len; i++){
    if(message[i] == ' '){
      new_message[i] = ' ';
      continue;
    }
    if(upper == 0){
      new_message[i] = tolower(message[i]);
      upper = 1;
    }
    else if(upper == 1){
      new_message[i] = toupper(message[i]);
      upper = 0;
    }
  }
  new_message[len] = '\0';

  return(new_message);
}
