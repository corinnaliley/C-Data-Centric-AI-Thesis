TASK:
Programmieren Sie eine Funktion `void magic(char *path, int number);`, die
einen String mit dem Pfad zu einer Datei `path` und einen ganzzahligen Wert
`number` zwischen `0` und `9` übergeben bekommt. In die Datei am Pfad `path`
soll ein String folgender Form geschrieben werden: Der String besteht aus
endlich vielen Satzteilen, die je mit einem Komma getrennt werden. Das erste
Wort eines Satzteiles ist das englische Zahlwort von `number` gefolgt von "is"
und dem englischen Zahlwort von *der Anzahl der Zeichen des Zahlwortes von*
`number`. Diese Teile werden solange gebildet und aneinander gereiht, bis `4`
(four) erreicht wurde, dann lautet der String "four is magic".


### Beispielaufruf
```
magic("beispiel_1.txt", 0);
magic("beispiel_2.txt", 1);
```
### Beispielergebnis
```
beispiel_1.txt: 
  zero is four, four is magic.
beispiel_2.txt: 
  one is three, three is five, five is four, four is magic.
```

### Hinweis
Benutzen Sie `<string.h>`, um geeignete Funktionen zur
Stringbearbeitungen zu benutzen.


SOLUTION:
#include <stdio.h>
#include <string.h>

char *names[10] = {"zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"};

void magic(char *path, int number) {
  FILE *fp;
  fp = fopen (path,"w");
  if(fp == NULL) {
    printf("Error on opening file");
    return;
  }
  char stringtoput[1000];
  char *is = "is";
  int count = number;

  while (count != 4) {
    strcat(stringtoput, names[count]);
    strcat(stringtoput, " ");
    strcat(stringtoput, is);
    strcat(stringtoput, " ");
    count = strlen(names[count]) ;
    strcat(stringtoput, names[count]);
    strcat(stringtoput, ", ");
  }
  strcat(stringtoput, "four is magic.");
  // printf("%s\n",stringtoput);

  for (int i = 0; i < strlen(stringtoput); i++) {
    fputc(stringtoput[i], fp);
  }
  fclose(fp);
}
