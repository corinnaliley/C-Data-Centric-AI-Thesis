Die <a target="_blank" rel="noopener noreferrer" href="https://de.wikipedia.org/wiki/Levenshtein-Distanz">**Levenshtein-Distanz**</a> zwischen zwei Worten gibt an, wie viele Zeichen man ersetzen, einfügen, oder löschen muss, um ein Wort in das andere zu verwandeln.
Schreiben Sie eine Funktion 

```
int levenshtein(char* wort1, char* wort2)
```

welche zwei Strings entgegennimmt und deren Levenshtein-Distanz zurückgibt. Verwenden Sie dazu den Algorithmus, der unter dem Link oben beschrieben ist.
 
*Anforderungen*
 
- Für zwei leere Strings gibt die Funktion 0 zurück.
- Das im Wikipedia-Artikel beschriebene Backtracing müssen Sie nicht durchführen.
 
*Hinweise*
 
- Verwenden Sie in dieser Aufgaben für das Feld dynamischen Speicher um dem Umgang damit zu üben. 
- Nutzen Sie dafür die Funktion `malloc()`.
- Vergessen Sie nicht den Speicher wieder freizugeben.
 
### Beispielaufruf
```
levenshtein("Tier", "Tor");
```

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int min(int a, int b, int c) {
  if (a <= b && a <= c) {
    return a;
  } else if (b <= a && b <= c) {
    return b;
  } else {
    return c;
  }
}

int levenshtein(char *wort1, char *wort2) {

  if (wort1 == NULL || wort2 == NULL)
    return 0;
  if (*wort1 == '\0' && *wort2 == '\0') {
    return 0;
  }
  int len1 = strlen(wort1);
  int len2 = strlen(wort2);

  unsigned int matrix[len1+1][len2+1];


  matrix[0][0] = 0;

  for (int j = 1; j <= len2; j++) {
    matrix[0][j] = matrix[0][j - 1] + 1;
  }

  for (int i = 1; i <= len1; i++) {
    matrix[i][0] = matrix[i - 1][0] + 1;
  }

  for (int i = 1; i <= len1; i++) {
    for (int j = 1; j <= len2; j++) {
      matrix[i][j] = min(matrix[i - 1][j] + 1, matrix[i][j - 1] + 1,
                    matrix[i - 1][j - 1] + (wort1[i - 1] == wort2[j - 1] ? 0 : 1));
    }
  }

  int v = matrix[len1][len2];
  return v;
}
