TASK:
1. Programmieren Sie eine Funktion `int f(int s, int n)`, die bei gegebenen ganzzahligem Startwert `s` und gegebenen Integer `n` das n-te Folgenglied der folgenden Folge F berechnet: `F[n] = F[n-1] + 2`

2. Programmieren Sie eine Funktion `int g(int n)`, die bei gegebenen Integer `n` das n-te Folgenglied berechnet. Das n-te Folgenglied ist die n-te Zweierpotenz um eins verringert.

3. Programmieren Sie eine Funktion `int h(int n)`, die bei gegebenen Integer `n` das n-te Folgenglied berechnet. Die Folge kann durch folgenden Algorithmus beschrieben werden: Das erste Folgenglied für `n = 1` ist die Quadratzahl `h(n) = 1`. Jedes weitere Folgenglied kann aus dem vorherigen berechnet werden, indem die jeweils nächste Quadratzahl addiert wird.

Achten Sie auf das Einbinden der richtigen header-Dateien.

### Beispielaufruf
```
f(1,8)
g(2)
h(3)
```
### Beispielergebnis
```
17
3
14
```


SOLUTION:
#include<stdio.h>
#include<math.h>

int f(int s, int n) {
  int sum = s;
  for (int i=0;i<n;i++) {
    sum += 2;
  }
  return sum;
}

int g(int n) {
  int sum = (int) pow(2, n);
  return (sum-1);
}

int h(int n) {
  int sum = 0;
  for(int i=0;i<n;i++) {
    sum += (int) pow(i+1, 2);
  }
  return sum;
}
