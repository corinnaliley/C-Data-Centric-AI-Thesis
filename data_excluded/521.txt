TASK:
Ein Alphabet ist eine Zeichenfolge in der kein Zeichen mehr als einmal vorkommt.

Ein Text über einem Alphabet ist eine Zeichenfolge, die nur aus Zeichen des Alphabets besteht.

Für zwei Alphabete mit derselben Anzahl an Zeichen, kann man einen Text über dem einen Alphabet (Quellalphabet) in einen Text über dem anderen Alphabet (Zielalphabet) übersetzen, indem man die Zeichen einzeln übersetzt.

Die Übsetzung eines Zeichens aus dem Quellalphabet ist das Zeichen aus dem Zielalphabet an derselben Position.

*Beispiel*

`1234567890` Quellalphabet

`0987654321` Zielalphabet

Der Text `149` über dem Quellalphabet wird zum Text `072` über dem Zielalphabet übersetzt.

Programmieren Sie eine Funktion mit Prototyp

```
char *substitute(const char *source, const char *target, const char *text);
```

und Parametern:

- `source`, ein Zeiger auf ein `char`-Feld in dem ein Quellalphabet, als String, gespeichert ist,
- `target`, einen Zeiger auf ein `char`-Feld in dem ein Zielalphabet, als String, gespeichert ist,
- `text` , einen Zeiger auf ein `char`-Feld in dem ein zu übersetzender String gespeichert ist.

Die Funktion substitute liefert einen Zeiger auf ein `char`-Feld zurück, in dem sich die Übersetzung des Strings `text` über dem Quellalphabet `source` in das Zielalphabet `target` beﬁndet.

*Anforderungen*

- Die Funktion liefert bei einem Fehler den Nullzeiger, sonst einen Zeiger auf den übersetzten Text zurück.
- Der Speicher für das Feld, in dem der übersetzte Text zurückgeliefert wird, wird auf dem Heap reserviert, dabei kann ein Fehler auftreten.
- Enthält der zu übersetzende Text Zeichen, die nicht im Quellalphabet vorhanden sind, ist das ein Fehler.
- Tritt ein Fehler auf nachdem Speicher auf dem Heap angefordet wurde, muss dieser Speicher vor dem Ende der Funktion wieder freigeben werden.
- Sie brauchen nicht zu prüfen, ob `source` und `target` auf den Anfang von `char`-Felder zeigen, ob diese Strings enthalten und ob in diese Strings gültige Alphabete gleicher Länge sind.
- Sie brauchen nicht zu prüfen, ob `text` auf den Anfang eines `char`-Feldes zeigen, das einen String enthält.
- Berücksichtigen Sie das Null-Zeichen bei der Ermittlung des Speicherplatzes für den resultierenden (übersetzten) String aus dem übergebenen (zu übersetzenden) String.


SOLUTION:
#include <string.h>
#include <stdio.h>
#include <stdlib.h>
  
char *substitute(const char *source, const char *target, const char *text){
    int length = strlen(text);
    char * result = malloc(sizeof(char) * (length +1));
    if (result == NULL)
        return NULL;
    for(int i = 0; i < length; i++){
        int flag = 0;
        for(int j = 0; j < strlen(source); j++){
            if( source[j] == text[i]){
                result[i] = target[j];
                flag = 1;
            }
        }
        if (flag == 0){
            free(result);
            return NULL;
        }
    }
    //result[length-1] = 'G';
    result[length] = '\0';
    return result;
}
