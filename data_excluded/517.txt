TASK:
Programmieren Sie eine Funktion, mit folgendem Prototyp

```
int is_perm(const int perm[], int length);
```

die testet, ob das Feld der Länge `length` , auf dessen Anfang `perm` zeigt, eine Permutation der Zahlen `1, 2, ..., length` beschreibt.

*Anforderungen:*

- Die Funktion liefert bei einem Fehler einen negativen Wert zurück.
- Handelt es sich um eine Permutation der Zahlen `1, 2, ..., length` wird ein positiver Wert zurückgeliefert, andernfalls wird `0` zurückgeliefert.
- Berücksichtigen Sie den Fall, dass perm Werte nicht aus `1, 2, ..., length` enthält.
- Testen Sie wie folgt.
  1. Speicher für ein `int`-Feld `anzahl` der Länge `length` auf dem Heap anfordern (hier kann ein Fehler auftreten).
  2. Für jeden Wert *k* aus `1, 2, ..., length` wird im Feld `anzahl` abgespeichert, wie oft *k* im Feld `perm` vorkommt.
  3. Enthalten alle Komponenten des Feldes `anzahl` den Wert 1 handelt es sich um eine passende Permutation.
  4. Vor Beendigung der Funktion wird der angeforderten Speicher wieder freigegeben.

### Beispielaufruf
```
int p1[] = {1, 3, 4, 5, 0};
is_perm(p1, 5)
```
### Beispielergebnis
```
0
```


SOLUTION:
#include <stdio.h>
#include <stdlib.h>

int is_perm(int perm[], int laenge) {

  int *anzahl;
  int wert = 1;
  int i;

  if (perm == NULL)
    return -1;

  /* Test auf wohldefiniert Eingabe , siehe unten */
  for (i = 0; i < laenge; i++) {
    if (perm[i] < 1 || perm[i] > laenge)
      return 0;
    /* Fehler waere auch OK */
  }

  if ((anzahl = malloc(laenge * sizeof(int))) == NULL)
    return -3;

  for (i = 0; i < laenge; i++)
    anzahl[i] = 0;

  for (i = 0; i < laenge; i++)
    ++anzahl[perm[i] - 1];

  for (i = 0; i < laenge; i++)
    wert = wert & (anzahl[i] == 1);

  free(anzahl);
  return wert;
}
